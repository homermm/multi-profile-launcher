# Lanzador de Múltiples Perfiles de Navegador

Este repositorio contiene un conjunto de scripts de Python para crear y lanzar múltiples perfiles de navegador de forma masiva. Es ideal para tareas que requieren sesiones de usuario separadas, como testing de aplicaciones web, gestión de múltiples cuentas sociales o cualquier actividad que se beneficie del aislamiento de perfiles.

## Navegadores Soportados

Este proyecto está organizado por navegador. Haz clic en la carpeta del navegador que te interese para ver sus scripts e instrucciones específicas.

* **[Firefox (para Linux)](./firefox/README.md)**
    * `crear_perfiles.py`: Script para crear perfiles en Firefox.
    * `lanzar_perfiles.py`: Script para lanzar los perfiles creados.

* **[Brave (para Windows)](./brave/README.md)**
    * `lanzar_perfiles.py`: Script único que crea y lanza perfiles de Brave.

## Requisito General

* **Python 3**: La mayoría de los sistemas Linux ya lo tienen. Para Windows, se puede descargar desde [python.org](https://www.python.org/).
